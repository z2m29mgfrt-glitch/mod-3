Ответы на контрольные вопросы по лабораторной работе 4:

1. jQuery - библиотека JavaScript, упрощающая работу с веб‑страницами. Позволяет компактно искать и менять элементы, обрабатывать события, выполнять AJAX‑запросы и анимации.

2. Подключить jQuery можно через CDN: добавить строку с библиотекой перед скриптом в конце <body>. Или локально: скачать jquery.min.js и указать путь в <script>.

3. Символ $ - псевдоним функции jQuery. Запускает работу библиотеки, нужен для поиска элементов, создания HTML‑элементов и операций, например $('#id') или $('<div>').

4. .text() работает с текстом элемента: HTML‑теги отображаются как текст. .html() работает с HTML‑разметкой: теги выполняются и отображаются по правилам HTML.

5. Найти элемент по ID в jQuery можно с помощью селектора #id. Например, $('#myElement') найдёт элемент с id="myElement".

6. Метод .val() читает или устанавливает значения элементов форм: <input>, <textarea>, <select>. Например, $('#myInput').val() вернёт введённый текст, $('#myInput').val('Текст') установит новое значение.

7. Добавить новый элемент в конец списка можно методом .append(). Например, $('#list').append($('<li>Новый пункт</li>')) добавит <li> в конец контейнера с ID list.

8. Делегирование событий - техника, когда обработчик вешают на родительский элемент, а не на каждый дочерний. Нужна для обработки событий на динамически создаваемых элементах и сокращения числа обработчиков.

9. Удалить элемент из DOM с помощью jQuery можно методом .remove(). Например, $('#element').remove() удалит элемент с ID element вместе с содержимым.

10. .parent() возвращает непосредственного родителя элемента. .closest() ищет ближайшего родителя (включая сам элемент), который соответствует заданному CSS‑селектору.

11. Перебрать элементы с определённым классом в jQuery можно циклом $.each(). Например, $('.myClass').each(function(index, element) { ... }) выполнит функцию для каждого элемента с классом myClass.

12. localStorage - механизм веб‑хранилища в браузере для сохранения данных на стороне клиента. Данные хранятся долго, пока пользователь не очистит кеш. Работать можно через методы: localStorage.setItem('ключ', 'значение') (сохранить), localStorage.getItem('ключ') (прочитать), localStorage.removeItem('ключ') (удалить).
